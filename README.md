<h2>Todo-list 만들기</h2>

**프론트엔드**: HTML, JavaScript, CSS, jQuery, Ajax를 사용했으며 Bootstrap
의 아이콘, 버튼을 활용함 (+ 반응형 구현)

**백엔드**: DB는 MongoDB 사용 서버는 Flask를 이용함

**기능 구현**: 남은 메모 수, 현재 날짜, 메모 추가 및 삭제, 메모가 없을때 나타나는 아이콘 등
